Deno markdown-wasm
==================

A markdown parser module with WebAssembly for Deno

# how to use?

```typescript
import {renderMarkdown} from "https://denopkg.com/linux-china/deno_markdown_wasm/mod.ts";

console.log(renderMarkdown("#h1"));
```
